## Projetos em Destaque  
  
### Novo Cortex  
Sistema backend institucional desenvolvido com Django + DRF para gerenciamento de identidade, vínculos organizacionais e relações acadêmicas.  
  
O projeto foi estruturado com foco em modularização de domínios, separação de responsabilidades e manutenção de regras de negócio complexas em ambiente institucional.  
  
#### Principais implementações  
  
- Arquitetura em camadas (`View → Business → Rules → Helpers`)  
- Estrutura modular baseada em domínios de negócio  
- Autenticação JWT com login via CPF  
- Sistema de permissões e gerenciamento de usuários  
- Organização institucional e vínculos acadêmicos  
- Documentação OpenAPI com Swagger/ReDoc  
- Auditoria de alterações com histórico automático  
- Base reutilizável de componentes internos (`AppCore`)  
- Padronização de tratamento de exceções e paginação  
- Estrutura preparada para expansão de bounded contexts  
  
#### Stack  
  
`Python` `Django` `Django REST Framework` `PostgreSQL` `JWT` `drf-spectacular` `django-allauth` `Docker`  
  
Repositório:  
- <https://github.com/soareslucas9090/novo_cortex>

### CertEthereum  
Sistema para emissão e validação de certificados acadêmicos utilizando blockchain Ethereum.  
  
Projeto desenvolvido com foco em integridade, rastreabilidade e descentralização de dados através de contratos inteligentes e APIs REST.  
  
#### Principais implementações  
  
- API REST para emissão e validação de certificados  
- Integração com contratos inteligentes via Web3.py  
- Autenticação e gerenciamento de permissões  
- Documentação OpenAPI/Swagger  
- Envio automatizado de certificados  
- Arquitetura modular para integração com outros sistemas  
  
#### Stack  
  
`Python` `Django` `DRF` `PostgreSQL` `Solidity` `Ethereum` `Web3.py`
